How to Clear Cache on Windows

When to Clear Cache?

  • Your computer is running slowly.
  • To free up storage space.
  • A website isn't displaying correctly.
  • Troubleshooting system or app issues.
  • To remove traces of your browsing activity.

Objective

To show you how to clear temporary system files and browser cache on Windows to improve performance, free up space, and resolve common issues.

  1. Navigate to Settings: Open the Windows Start menu, type "Settings", and open the app.
  2. Go to Storage: In Settings, navigate to System > Storage.
  3. Find Temporary Files: Click on Temporary files. Windows will take a moment to scan your system.
  4. Select and Remove: Check the boxes for the file types you want to delete (e.g., Temporary Internet Files, Recycle Bin, Thumbnails). Be careful with the 'Downloads' folder if you have files there you want to keep.
  5. Confirm Deletion: Click the Remove files button to clear the selected data.

These general steps apply to most modern browsers like Chrome, Firefox, and Edge.

  1. Open Browser Settings: Launch your browser. Click the three-dot menu icon (usually in the top-right corner) and select Settings.
  2. Find Privacy Options: Look for a Privacy and security section in the settings menu.
  3. Clear Browsing Data: Click on Clear browsing data (or a similar option).
  4. Choose Data and Time Range: In the dialog box, select a time range like "All time". Ensure that "Cookies and other site data" and "Cached images and files" are checked.
  5. Confirm Action: Click the Clear data button to finalize.

Quick Keyboard Shortcut

For most browsers, you can press Ctrl + Shift + Delete to directly open the "Clear browsing data" window.

Process Complete

You have successfully cleared the cache.
